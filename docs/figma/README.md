## 如何自行拼接链接

同一文件内换节点时，可替换查询参数中的 `node-id`（Figma 用 **连字符**：`8111:3433` → `node-id=8111-3433`）：

```text
https://www.figma.com/design/<FILE_KEY>/<文件名>?node-id=<NODE_ID>
```
